GenOtoScope will annotate the variant using open bioinformatic databases and tools and will predict its ACMG pathogenicity class and posterior probability. Also comments for each examined ACMG rule will be provided to help user jugde the prediction. ..

Predict hearing-loss pathogenicity for whole exome file?

For data protection reasons, we allow online users to classify only one genomic variant using GenOtoscope. If you would like to annotate a whole exome sequencing file (WES) please install and run the command line version of the tool.
